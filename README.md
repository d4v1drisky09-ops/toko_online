# Mini E-Commerce Sederhana

Aplikasi toko online sederhana dengan PHP Native + MySQL.

## Fitur

- Login & Register User
- Dashboard Admin (CRUD Produk)
- Daftar Produk
- Keranjang Belanja
- Checkout dengan update stok otomatis
- Riwayat Transaksi

## Cara Install

1. Buat database `toko_online` di MySQL
2. Import file `database.sql`
3. Akses melalui browser: http://localhost/toko_online

## Akun Default

- Admin: admin / admin123
- User: user / user123
